# v0.2.0 · Ambient memory-graph — EXPLORE brief

> *Stit'Claude · Phase 1. Written before a line of code.*

## Intent

```
BUT         : Rendre le composant *ambiant* — il track et surface le
              parcours de l'utilisateur à travers un site/app multi-écrans,
              pas juste un seul article.
AUDIENCE    : Devs React qui veulent un "reading graph" global (SaaS,
              dashboard, essai multi-chapitre, extension Chrome).
SUPPORT     : Web, inclut multi-page SPA via History API.
HOUSE       : Editorial × Power-Dense (continuité avec v0.1.0).
RÉFÉRENCES  : Linear (mono kickers, hairlines denses) · NYT editorial
              (rythme temporel) · Stripe docs (colonnes de contexte).
CONTRAINTES : Non-breaking · lib générique (pas de site-creep) ·
              tokens OKLCH only · ≤300 lignes/fichier · loi 3 (Y=firstAt)
              préservée · reduced-motion safe.
ANTI-GOALS  : Pas de kinds site-spécifiques · pas de dépendance à un
              routeur · pas de filter pills overload · pas de bento égal.
```

## Nouvelles API publiques

### Lib

| Surface | Ajout |
|---|---|
| `Node.route?: string` | Abstract route bucket. Le consumer passe ce qu'il veut (pathname, tabId, docId, mode…). |
| `<Root route?: string>` | Route courante. Stockée au commit time sur chaque nouveau node. |
| `<Graph renderNode?>` | Escape hatch pour custom node shapes par id ou kind. Signature : `(item, ctx) => ReactNode \| null`. Return `null` = fallback default. |

### Règles de rendu

- **Colonnes** · activées quand `nodes.some(n => n.route) && uniqueRoutes ≥ 2`. Sinon single-column classique (backward-compatible).
- **Largeur colonne** · 180px min. Stack per-column pour `MIN_NODE_SEPARATION_Y` (pas de push-down inter-colonne).
- **Label** · mono uppercase, 10px, letter-spacing 0.22em, sticky top via HTML overlay (pas SVG). Coral sur route courante, muted ailleurs.
- **Séparateurs** · 1px dashed vertical, opacity 0.3, entre colonnes.
- **Tint route courante** · `color-mix(in oklch, var(--mg-accent) 3%, transparent)` sur toute la colonne.
- **Route-jump edges** · attribut `data-mg-route-jump` sur forward/return edges crossant des routes. CSS distinct (coral subtil 0.8px vs fg-subtle 1px).
- **Return arc** · loi 4 préservée (curve right, même control point formula).

### Scroll

- Panel body · `overflow-y: auto` (existant)
- Graph wrap · `overflow-x: auto; overflow-y: visible` (NEW)
- `scroll-snap-type: x proximity` + `scroll-snap-align: center` sur colonnes
- Auto-follow · `route` prop change → `scrollTo(left: columnX - halfWidth, smooth)`
- Scrollbar thin, coral-tinted au hover

## Token system — DISTILL

Aucun nouveau token. Extension seulement :
- `color-mix(in oklch, var(--mg-accent) 3%, …)` — tint route courante
- `color-mix(in oklch, var(--mg-fg) 30%, var(--mg-border))` — séparateurs
- Tous les motion · `--mg-duration-*` + `--mg-ease-*`
